# epireview: Tools to update and summarise the latest pathogen data from the Pathogen Epidemiology Review Group (PERG)

Contains the latest open access pathogen data from the Pathogen
Epidemiology Review Group (PERG). Tools are available to update pathogen
databases with new peer-reviewed data as it becomes available, and to
summarise the latest data using tables and figures.
