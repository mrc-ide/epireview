Quality assessment questionnaire This function returns the list of 7
questions used to assess quality of articles.

## Usage

``` r
qa_questions()
```

## Value

a data.frame with the following columns: (a) qnames: these are the names
of the corresponding columns in the articles data.frame; (b) qtext: the
text of the question, and (c) notes: any additional notes.
